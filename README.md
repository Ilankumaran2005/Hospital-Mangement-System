# Hospital Management System

A web-based Hospital Management System developed using PHP and MySQL to manage patient records efficiently through CRUD operations.

## Features

* Add new patient records
* View patient details
* Update patient information
* Delete patient records
* Responsive user interface using Bootstrap

## Technologies Used

* PHP
* MySQL
* HTML5
* CSS3
* Bootstrap 5
